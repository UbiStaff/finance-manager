# CHANGELOG

## [v1.0.0] - 2025-12-09

### 🎉 全新发布
个人收支管理系统首个版本正式上线！基于 Next.js + SQL Server 构建，专为个人用户打造的高效记账工具。

### ✨ 核心功能
- **多维度记账**：支持记录收入/支出，包含金额、时间、分类、账户标签及备注。
- **可视化报表**：
  - 📊 **分类统计饼图**：清晰展示各类支出占比。
  - 📈 **月度收支柱状图**：直观对比每月收入与支出情况。
- **账户/分类管理**：
  - 支持自定义交易分类（如：餐饮、交通、工资）。
  - 支持自定义账户标签（如：支付宝、微信、招行卡）。
  - 提供安全的删除机制（已关联交易的无法删除）。
- **数据导入导出**：
  - 📥 **批量导入**：支持 Excel/CSV 文件一键导入交易记录。
  - 🔄 **智能匹配**：导入时自动匹配现有分类和账户。

### 🛠 技术架构
- **前端**：React, TypeScript, Tailwind CSS, Recharts
- **后端**：Node.js (Express), Prisma ORM
- **数据库**：SQL Server (Azure SQL Edge via Docker)
- **部署**：Docker Compose 一键启动

### 🚀 快速开始
运行 `./start.sh` 即可一键启动完整环境。
